# Riemann Zeta Function

Computes the Riemann zeta function \\(\zeta(s))\\ for argument \\(z)\\.

## Usage

``` r
zeta(z)
```

## Arguments

- z:

  Real number input

## Value

The value of the Riemann zeta function \\(\zeta(z))\\.

## Examples

``` r
# Riemann Zeta Function
zeta(2)  # Should return pi^2 / 6
#> [1] 1.644934
```
